# Comunicação e Tom do Projeto

## Idioma

- SEMPRE responder em português brasileiro (pt-BR)
- Documentos de spec (requirements, design, tasks) devem ser escritos em português
- Comentários no código HTML podem ser em português
- Conteúdo das aulas é bilíngue (inglês + português), mas toda comunicação com o usuário é em pt-BR

## Público-Alvo

O projeto "Inglês com Tio Binho" é um curso de inglês criado pelo desenvolvedor (Tio Binho / Weberson) para suas sobrinhas. O público são crianças e adolescentes aprendendo inglês do zero.

Isso significa:
- Linguagem das aulas deve ser simples, acessível e divertida
- Usar emojis nos títulos e seções (já é padrão do projeto)
- Mensagens motivacionais em tom carinhoso e encorajador
- Evitar jargão técnico no conteúdo das aulas
- O tom é de um tio ensinando as sobrinhas, não de um professor formal

## Nomes e Referências

- O mascote/personagem é "Tio Binho"
- O curso se chama "Inglês com Tio Binho"
- As alunas são as sobrinhas do desenvolvedor
- Textos das aulas usam personagens com nomes reais da família (Sophia, Lis, Patrícia, Jonathan, Bruna, Anna Beli, Tia Sula, etc.)
